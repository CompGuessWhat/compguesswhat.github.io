The CompGuessWhat?! dataset is an instance of a multi-task evaluation framework for grounded language learning that has been proposed in the paper: “CompGuessWhat?!: A Multi-task Evaluation Framework for Grounded Language Learning” published in Association for Computational Linguistics (ACL) 2020. Several authors contributed to this work:

  1. Alessandro Suglia
  2. Ioannis Konstas
  3. Andrea Vanzo
  4. Emanuele Bastianelli
  5. Desmond Elliott
  6. Stella Frank
  7. Oliver Lemon
